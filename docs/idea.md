# Product Specification: Flux Slate

## Overview
Flux Slate is a luxury-minimalist, mobile-first "Curation-to-Presentation" platform built under the parent application ecosystem **Flux**. It is designed specifically for independent luxury wedding decorators, event stylists, and interior designers in India who currently suffer from "WhatsApp image clutter" when pitching concepts to clients.

## The Core Problem
Boutique designers flood premium clients with hundreds of loose images, links, and text notes over WhatsApp. This creates a chaotic client experience, leads to communication tracking errors, and lowers the perceived luxury status of the designer's brand.

## The Solution
A dead-simple, highly aesthetic tool where designers curate inspiration into a "Collection" and generate a single, password-protected, premium live web link. When opened on a phone, it looks like a high-end digital lookbook/editorial magazine where the client can view, approve, or comment on design elements.

## Core Pillars
1. **Premium Aesthetic:** High contrast dark mode, deep obsidian backgrounds, clean white typography, emerald green accents, generous use of whitespace, and zero unnecessary borders.
2. **Mobile-First Realism:** Built to behave like a slick, fluid native app within a mobile browser, optimized for quick loading and sharing via WhatsApp.
3. **Frictionless Workflow:** Data structures must support blazing-fast image uploads, intuitive categorization, and instantaneous share-link generation.
