# Phase 1: Validation Playbook

## Objective
To capture 15 high-intent email/Instagram signups from independent Indian event decorators within 7 days using zero marketing budget.

## Execution Steps
1. Deploy the `/landing` folder containing the elegant single-page lookbook teaser.
2. Ensure the form fields perfectly capture: `Full Name`, `Instagram Handle`, `Business Email`.
3. Set up the submission workflow to route entries securely.

## Metric for Success
- If Signups >= 15 -> Proceed to unlock code generation instructions for `/app`.
- If Signups < 15 -> Refine marketing copy on the landing page, optimize value proposition hooks, or re-verify target handles on Instagram. Do NOT write `/app` backend architecture until this phase passes.
